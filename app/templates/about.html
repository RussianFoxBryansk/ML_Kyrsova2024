Сведения о сетях

В данном веб-приложении представлены модули с обученными нейронными сетями, объединённые общей темой "Интернет-магазин".

Модели

  • Модель 1: CNN Model

    Эта модель основана на сверточной нейронной сети (CNN) и предназначена для классификации изображений. Она обучена на наборе данных, содержащем различные категории товаров, такие как одежда, обувь, косметика и электроника.

    Параметры:

    • Размер входного изображения: 64x64 пикселей
    • Количество классов: {{ class_names|length }}

  • Модель 2: Dense Model

    Эта модель представляет собой полносвязную нейронную сеть (Dense) и предназначена для классификации изображений. Она может использоваться для более глубокого анализа изображений и выявления более сложных паттернов.

    Параметры:

    • Размер входного изображения: 64x64 пикселей
    • Количество классов: {{ class_names|length }}

  • Модель 3: YOLOv5

    Общая информация о процессе обучения
    • Общее время обучения: 50 эпох завершено за 48.385 часов.
    • Размер модели: Оптимизатор был удален из весов, чтобы уменьшить размер файлов:
      • `last.pt`: 15.0 MB (веса последней эпохи)
      • `best.pt`: 15.0 MB (веса с наилучшими результатами на валидации)
    Валидация модели

    После завершения обучения модель была проверена на валидационном наборе данных:

    • Сводка модели:
      • Количество слоев: 157
      • Количество параметров: 7,279,825
      • GFLOPs: 16.6 (это мера вычислительной сложности модели)
    Результаты валидации

    Валидация проводилась на 652 изображениях с 971 экземпляром объектов. Результаты представлены в виде различных метрик:

    • P (Precision): Точность, показывающая долю истинных положительных результатов среди всех положительных предсказаний.
    • R (Recall): Полнота, показывающая долю истинных положительных результатов среди всех реальных положительных объектов.
    • mAP50: Средняя точность при IoU (Intersection over Union) 0.5.
    • mAP50-95: Средняя точность при IoU от 0.5 до 0.95.

Документация API

Ниже представлена документация по API, доступному в данном приложении.

1. Получение списка всех моделей

Метод: GET

URL: /api/models

Описание: Получает список всех доступных моделей нейронных сетей.

Пример запроса:

GET /api/models

2. Классификация изображения с помощью модели CNN

Метод: POST

URL: /api/classify/cnn

Описание: Отправляет изображение для классификации с использованием модели CNN.

Параметры:

  • image: Изображение в формате base64.

Пример запроса:

POST /api/classify/cnn
{
    "image": "data:image/jpeg;base64,..."
}

3. Классификация изображения с помощью модели Dense

Метод: POST

URL: /api/classify/dense

Описание: Отправляет изображение для классификации с использованием модели Dense.

Параметры:

  • image: Изображение в формате base64.

Пример запроса:

POST /api/classify/dense
{
    "image": "data:image/jpeg;base64,..."
}

4. Обработка изображения с помощью YOLOv5

Метод: POST

URL: /api/detect/yolov5

Описание: Отправляет изображение для обработки с использованием модели YOLOv5.

Параметры:

  • file: Изображение в формате multipart/form-data.

Пример запроса:

POST /api/detect/yolov5
--data-binary @path_to_your_image.jpg
